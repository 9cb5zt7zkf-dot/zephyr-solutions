# Zephyr Solutions — Reconstructed Site (Backup / Starting Point)

This is a from-scratch, hand-coded reconstruction of zephyrsolutions.ae, built as a backup / starting point per your request.

## What this is
- Plain HTML5 + CSS3 + vanilla JS — no framework, no build step. Open any `.html` file directly or serve the folder with any static host.
- 6 pages: Home (`index.html`), About, Services, Licensing, Pricing, Contact.
- Navy/gold design system matching the original site's positioning (UAE business setup / company formation consultancy).
- Content (services, pricing packages, licensing comparison, contact details) was reconstructed from what could be read off the live site's pages at the time of writing.
- Contact form uses a `mailto:` link (opens the visitor's email app pre-filled) — no backend required. WhatsApp click-to-chat button included as well.

## Important limitations — please review before publishing
Because this was built as a faithful re-creation rather than a byte-for-byte copy (no direct file/asset scraping was possible in this environment), a few things are worth double-checking against the real site before this goes live anywhere:

- **No real photos or official logos.** All imagery (skyline graphic, icon mark, wordmark logos) is hand-drawn SVG, not the original site's actual photos or government/authority logos. Swap in real photography and any officially licensed logos before publishing.
- **Prices and package details** (Freezone Sharjah/Dubai/Ajman, Mainland Dubai figures) were extracted from the live pricing page as read at build time — confirm these still match your current published rates before this goes anywhere public, since pricing pages change.
- **Contact details** (address, phone, email, hours) were carried over as read from the site — please verify these are still current.
- **Copy/wording** is a rewritten reconstruction of the same information, not a literal copy of the original text.
